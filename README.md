x264-static
===========

> Static binaries for the `x264` command.

## Installation

```sh
$ npm install x264-static
```

## Usage

```js
const x264 = require('x264-static')
console.log(x264.path)
// '/home/werle/repos/little-core-labs/x264-static/bin/linux/x64/x264'
```
